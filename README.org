*** Работа с Lean
Есть два способа:
**** Установка Lean
https://lean-lang.org/install/

Нужно установить программу VS Code, затем в ней расширение Lean 4, которое после установки задаст несколько вопросов и установит Lean.  Этого должно быть достаточно.  Если будут проблемы, будем разбираться.
**** Работа online
+ https://lean.math.hhu.de/ 
+ https://live.lean-lang.org/ — последнюю неделю что-то барахлил, но сейчас, вроде, работает.
